What is the Internet?

To put it very simply, the Internet is a "network of networks". The different networks are connected, and they all communicate. This allows data to travel between computers. It is based on the TCP/IP protocol, which sends packets of data between different computers and different networks. To put it in a way that your dad would understand, the Internet is the roads in a country, sending trucks everywhere, each with different loads, going to different locations.

What is the World Wide Web?



The World Wide Web (WWW) is a method of accessing information using the Internet. The WWW mainly uses one set of "rules" to transfer information, known as HTTP. HTTP is Hyper Text Transfer Protocol, with a secure version called HTTPS. The most common use of HTTP is where networks transfer data, normally from websites, to your computer.

And all that Jazz.

Whilst the World Wide Web is a method of accessing information over the Internet, the Internet is the backbone of the World Wide Web. As we said before, the Web uses HTTP to transfer data, but the internet uses many more protocols, such as; SMTP, POP3 and IMAP (for eMail), FTP (for file transfer) and USENET (for news). As you can see, the Internet is more than just the World Wide Web. However, both of these have had a huge effect on our society as a whole.
